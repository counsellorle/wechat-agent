一、首次使用流程
────────────────────────────────────────

第 1 步：激活虚拟环境
  打开终端（Win+R → cmd），执行：
    F:\Ai\wechat-agent\venv\Scripts\activate

  终端前面出现 (venv) 即成功。

第 2 步：导出微信聊天记录为 JSON（三选一）

  【方式一】PyWxDump（推荐，一键导出）
    在 venv 激活状态下执行：
      pip install pywxdump
      pywxdump
    按提示操作，选择要导出的聊天对象，自动输出 JSON。
    将导出的 JSON 文件放入 data\raw\

  【方式二】WeChatMsg 工具
    下载 WeChatMsg（开源，GitHub 可搜），
    连接微信后导出聊天记录为 JSON，放入 data\raw\

  【方式三】手动构建
    按以下格式手动创建 JSON 文件放入 data\raw\：
    [
      {"content": "在吗？", "sender": "好友昵称", "type": "text", "timestamp": 1700000000},
      {"content": "在的",  "sender": "我",         "type": "text", "timestamp": 1700000010}
    ]
    参考文件：data\raw\sample_format.json（已自动生成）

第 3 步：清洗数据
    python src\02_clean.py
  输入你的微信昵称，程序会自动处理。

第 4 步：构建训练数据集
    python src\04_build_dataset.py

第 5 步：LoRA 微调训练（约 30 分钟）
    python src\05_train.py

第 6 步：构建 RAG 向量索引
    python src\03_build_rag.py

第 7 步：启动对话界面
    python src\06_chat.py

  浏览器自动打开 http://localhost:7860


二、日常对话（已有训练好的模型）
────────────────────────────────────────

  每次启动：
    F:\Ai\wechat-agent\venv\Scripts\activate

  浏览器打开 http://localhost:7860


三、增量重训（积累 50 条新对话后）

六、注意事项
────────────────────────────────────────

  · 全部本地运行，不需要联网
  · 对话记录自动保存在 chat_history.db，永不丢失
  · 模型文件约 3GB，占用 F 盘空间
  · 如需清空对话记忆，删除 db\chat_history.db 即可
  · 修改 config.py 中的 GRADIO_PORT 可更改端口号
